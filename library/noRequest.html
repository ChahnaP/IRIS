IRIS

API Endpoint

This is the API endpoint for the IRIS toolkit.

If you are attempting to send a request, you must send a POST request to this URL with the xmldata parameter equal to your XML data

If you are just starting out, you can try sending requests with our easy to use online requester.

If you want to register for a private client id visit our registration page.

More information on the formats of requests be found on our GitHub documentation.
